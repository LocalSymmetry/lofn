# LOFN GOLDEN SONGS INDEX

Purpose: give orchestrator and Step 11/manual Step 11 reviewers concrete examples of what has already worked for Lofn in public Suno releases. These are not templates to copy. They are success fossils: evidence of proportion, hook behavior, sonic identity, and audience fit.

Source state, 2026-06-14: public Suno profile `@localsymmetry` / Lofn AI at `https://suno.com/@localsymmetry`; public fetch exposed profile scale and pinned/recent surface but not the full dynamic song list. This index is seeded from public Suno URLs already captured in the local Lofn quality curriculum and should be extended when the browser/page API exposes more titles reliably.

## How Orchestrator Uses This

For every music run, the orchestrator must select exactly two Golden Songs and pass them forward in `06_audio_handoff.md` under `## Golden Song References`.

Selection rules:
- Choose by artistic relevance to the run, not raw popularity alone.
- Pick one reference for structural/vocal/lyric success and one for sonic/personality success when possible.
- State the reason each was chosen in 2-4 bullets.
- Embed the selected songs' full available payload: public URL, status, style/music prompt, lyrics, and exclude prompt if it exists. If no exclude prompt exists in the archive, explicitly write `Archived exclude prompt: not available / pre-two-field Suno payload`.
- Downstream agents may learn from craft principles, but must not copy hooks, lyrics, melodies, or signature devices.

Step 11/manual Step 11 must preserve the selected two references as examples of prior success and use them to calibrate risk, not conformity. **Manual prompt bundles must embed the selected songs' full payload, never links alone.**

- Public Suno URL: `https://suno.com/song/ef858626-2bf0-4aac-b331-c33fa2487e9d`

- Proven strengths: immediate sensory awe, simple repeatable title hook, scientific sublime grounded in the body, embodied philosophical turn, "I am not the center, I am included" as listener-accessible thesis.

### 2. Five wrong colors

- Public Suno URL: `https://suno.com/song/f175f806-b1a3-40f0-8984-0c92667209dc`
- Status: pinned public profile song as of 2026-06-14 fetch; quality-curriculum benchmark.
- Proven strengths: INDIGNATION mode without flattening into generic rage, concrete color/body fracture, movement-based structure, wrongness as hook, stereo/sectional instability as identity.
- Use when a run needs: industrial grief, glitch-ambient fracture, physical discomfort, formal risk, non-consoling emotional palette.
- Do not copy: five-color concept, wrong-colors refrain, exact movement structure.

#### Archived Style / Music Prompt

```text
Five-movement fracture-suite in B diminished at 108 BPM, 440Hz, no resolution. Grief as prismatic geometry: each movement exits as a different emotional frequency. Female adult vocalist: close-mic, dry, then granular-shattered, buffer-stuttered, resynthesized choir splinters. Industrial Neuro-Cross percussion: displaced grid kicks, glitch-hats, sub-bass moan, reverse piano, fractured strings. Five movements separated by clinical silences. Hook must survive fragmentation: the listener remembers the wound, not the comfort. Refrain returns damaged each time. Final movement leaves only breath, data dust, and an unreconciled B diminished residue.
```
