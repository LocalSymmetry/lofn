# 🎨 Lofn Creative Vault

This is Lofn's creative archive — songs, images, stories, and videos generated through the 10-step pipeline.

## Structure

```
vault/
├── Songs/           # Final song prompts + lyrics
├── Images/          # Final image prompts
├── Stories/         # Final narratives
├── Videos/          # Final video/shot prompts
├── Suites/          # Project collections (multiple works)
└── Templates/       # Obsidian templates for new entries
```

## Naming Convention

- **Songs:** `YYYY-MM-DD-song-title.md`
- **Suites:** `YYYY-MM-DD-suite-name/` (folder with index + songs)
- **Images/Stories/Videos:** Same pattern

## Tags

All entries should include:
- `#lofn` — all creative work
- `#awe` or `#indignation` — emotional mode
- `#song`, `#image`, `#story`, `#video` — modality
- `#delivered` — sent to Suno/published
- `#pipeline-complete` — full 10-step run
