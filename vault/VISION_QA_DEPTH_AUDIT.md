# Vision QA Depth Audit — Mandatory Gate

Created: 2026-06-01 | Status: ACTIVE
Parent: `vault/QA_DEPTH_AUDIT.md` — inherits all music QA learnings

**Lesson from music:** QA previously passed pipelines while Steps 03-05 were stubs and pair files were 46 lines each. This must not happen in vision.

---

## The Rule

**QA must audit LINE COUNTS and CONTENT DEPTH, not just file presence.** A file existing with 3 lines of filler is a FAIL, not a PASS.

## AUTO-FAIL Triggers — Image

Any of these = **immediate FAIL, do not pass go:**

1. Any step file under 10 lines (stub/filler)
2. Step 03-05 under half the minimum line count
3. Pair files under 25 lines (means no real prompts)
4. Fewer than 24 final prompts across all pairs
5. Any prompt under 80 words
6. Prompts missing ≥2 of the 7 required elements (emotional seed, medium as narrative agent, material specificity, lighting specification, three-tier focal hierarchy, chromatic storytelling, narrative incompleteness)
7. Same prompt text repeated across pairs without material variation
8. Template placeholders: "Artifact N", "Prompt N", "Concept N", lorem ipsum, TODO, TBD

---

## Prompt Density Checklist (per-prompt)

Each prompt must contain ALL 7 elements. QA must spot-check at least 3 prompts per pair:

- [ ] Emotional seed (first sentence: what feeling?)
- [ ] Medium as narrative agent (Polaroid flash, VHS still, oil impasto…)
- [ ] Material specificity (named surfaces: black glass, shag carpet, smoked crystal…)
- [ ] Lighting specification (named type: on-camera flash, CRT phosphor, sodium-vapor…)
- [ ] Three-tier focal hierarchy (primary/secondary/tertiary explicitly named)
- [ ] Chromatic storytelling (specific palette: harvest gold, apricot-white, sickly cyan…)
- [ ] Narrative incompleteness (unanswered question or unresolved event)

## VISUAL SOMATIC GATE

**Adapted from music Somatic Gate.** The 3 Hyper-Skeptics vote as a bloc on each step10 package:

1. **Concept Hyper-Skeptic** — *"Does this image hit the eye, or does it only read well as a description? Can Flux actually render this, or is it a mood board?"*
2. **Medium Hyper-Skeptic** — *"Is the medium choice actually doing work, or is it decorative? Would this be visually distinctive at thumbnail size?"*
3. **Context & Marketing Hyper-Skeptic** — *"Does this look like Lofn, or like any competent Midjourney prompt? Would someone stop scrolling?"*

**2 of 3 NO = BLOCKED.** The veto question: *"Is this visually distinctive enough to be Lofn, or could this have been generated by any competent prompt writer?"*
