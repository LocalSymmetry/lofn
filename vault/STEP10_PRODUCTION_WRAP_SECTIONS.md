# STEP10 PRODUCTION WRAP — Required Sections

Every pair agent (steps 06-10) MUST produce a step10 file containing ALL of the following sections. Do not abbreviate. Do not summarize. This is a production contract, not a summary.

## Mandatory Section List

1. **Step Provenance** — Step file template used, active personality, date, pipeline name, core constraint
2. **Hook Note** — The song's signature sonic device: what it is, when it appears, why it's unmistakable
3. **Active Personality Note** — Which personality is active, what mode (AWE/INDIGNATION), what register
4. **Continuity Payload Used** — What upstream context was consumed (golden seed, vocal identity, production environment, pair handoff)
5. **Producer-Grade Suno Music Prompt** — Dense prose paragraph, structural cues, NO genre names, NO artist names, follows 7-position order, <=1000 chars
6. **Categorized Negative / Avoid Prompt** — What Suno must NOT do, organized by category (vocal, rhythmic, textural, structural, genre)
7. **Public Lyrics** — Clean lyrics with EMO section headers, theme line, song form line
8. **Suno Lyrics** — Lyrics with embedded production cues, EMO tags, vox tags, prod tags
9. **Vocal Fingerprint** — Tessitura (specific note range), timbre, register, texture, mic technique, spatial treatment, mode-specific variations
10. **Style-Axis Lock** — Tempo, mood, instrumentation complexity, lyrical depth, genre fusion, vocal style, rhythmic complexity, melodic emphasis, harmonic richness, production style — each with a locked value
11. **Arrangement Dramaturgy** — Section-by-section breakdown with bar counts, energy states, spatial staging, somatic activity
12. **Production Dramaturgy** — Mix specifications: spatial architecture, frequency allocation, dynamic range, compression strategy, reverb strategy (NONE for vocals)
13. **Image Ladder Audit** — Verification that every lyric line has concrete sensory imagery, no abstractions without anchors
14. **Controlled Fracture** — Where the song breaks and how: silence placement, somatic cut, undecidable element activation
15. **Ghost Verse Bank** — Alternative verses, cut sections, B-side material for the ambitious arm
16. **Panel Ledger** — Which experts contributed, key debates, dissenting votes, hyper-skeptic veto record
17. **QA Pre-Flight Report** — Self-audit: all 7 production mandates verified, somatic gate pass, forbidden substitution sweep, undecidable element present, no artist names

## Size Expectation

A complete step10 production wrap is **8-40KB** (200-1000 lines). If your step10 output is under 5KB, you have missed sections. The minimum viable step10 is ~8KB with all 17 sections addressed.

## Usage

Inject this section list directly into every pair agent spawn task. Do NOT just say "step10_revision_synthesis.md" — list the sections explicitly.

## Reference

Gold-standard reference: `output/daily/2026-06-10/daily-run/audio/pair_01/pair_01_step10_revision_synthesis.md` (Fable 5 Ceremony, 40KB)
