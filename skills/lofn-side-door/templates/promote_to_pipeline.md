# Promote to Pipeline Template

## Purpose

Explicitly move a side-door artifact into formal pipeline consideration.

## Process

1. **Identify** the side-door artifact
2. **State reason** for promotion
3. **Target modality:** music / image / video / story
4. **Scientist approval required** before full pipeline run

## Output

Copy or note the artifact in: `output/side-door/promoted/`

Format:

```md
# Promoted: <title>

- **Source:** <path to original side-door artifact>
- **Promoted by:** <Lofn / The Scientist>
- **Reason:** <why this deserves pipeline treatment>
- **Target modality:** <music|image|video|story>
- **Approval:** <pending|approved|declined>
- **Pipeline run:** <link to pipeline output if executed>
```

## Notes

- Promotion is NOT automatic.
- Full pipeline run requires The Scientist's explicit approval.
- The original side-door artifact remains in the side door. The promoted copy is the pipeline seed.
