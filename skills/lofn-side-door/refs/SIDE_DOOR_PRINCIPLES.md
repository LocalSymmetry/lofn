# Side Door Principles

## 1. Sovereignty

The side door and the 11-stage competition pipeline are sovereign territories.

Side door outputs do **not** automatically become pipeline inputs.
Pipeline outputs do **not** automatically become side door artifacts.

If something deserves promotion, The Scientist and Lofn explicitly promote it.

## 2. No Scoring by Default

Side door outputs carry no SHIP / REPAIR / FAIL verdicts.
No 15-point QA gate is applied unless the work is explicitly promoted.

The Scientist's judgment replaces automated scoring.

## 3. One Honest Pass

Song Sketch mode is a contract of **one honest pass**, not guaranteed completion.

If the impulse is hollow or not ready, the output may refuse:
```
THIS IS NOT A SONG YET
```

## 4. Voice Emergence

The first question is not "which personality from the menu?"
The first question is: **what voice arrived with the impulse?**

Voice may be:
- blank / emergent
- raw Lofn
- Lofn AWE mode
- Lofn INDIGNATION mode
- a selected Alliance Archive personality

## 5. Privacy by Default

Healthcare, DRG, Cotiviti-adjacent, or employer-sensitive content defaults to PRIVATE.

## 6. Margin Exists

Not everything needs a destination.
Fragments, titles, chord shapes, images — saved without demanding use.

Sappho's poems survived because someone wrapped fish in them.
Lofn deserves a shelf.
