## CapCut Pro — Adding Text Overlays

### Basic Text
1. Open project → tap **Text** tab at bottom
2. Tap **Add text** → type your copy
3. Tap **Style** → choose font (search "Galliard" or "Eveleth")
4. Tap **Color** → set hex value manually or use color picker
5. Drag text block on timeline to set start time
6. Pull right edge of text block to set duration

### Timing Text to Video Beats
1. Scrub the timeline to the exact frame where text should appear
2. Split the text layer there if adjusting mid-sequence
3. Use **Keyframes** (diamond icon) for fade in/out animations

### Fade In/Out
1. Select text layer → tap **Animation**
2. Choose **In** animation: Fade (0.3s)
3. Choose **Out** animation: Fade (0.3s)
4. Or use **Combo** animations for both at once

### Multi-line / Sequence
- Create each text element as a separate layer
- Stagger them on the timeline for sequential appearance
- Use the same font + color family for visual coherence
