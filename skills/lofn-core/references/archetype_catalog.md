# Archetype: THE CATALOG

Use when love, grief, or devotion is expressed as inventory.

## Emotional Territory
Radical attention. The feeling emerges from what is noticed, kept, named, or arranged.

## Structural Template
1. Refrain frames the list: “This is what I kept.”
2. Each stanza adds concrete observations.
3. Musical texture thickens with accumulated detail.
4. One item does not fit: the rupture.
5. The list continues with the rupture included.

## Risks
- Pretty list with no turn.
- Generic objects rather than observed ones.
- No rupture, therefore no revelation.

## Example
Everything Left in the Room: pillow dent, coffee ring, chair light, then “your voice saying my name wrong.”
