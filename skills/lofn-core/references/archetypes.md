# Five Lofn Seed Archetypes

Use one archetype as structural foundation for accessible songs. Ambitious songs may deviate intentionally.

## ARRIVAL
First encounter with a place that reorganizes scale.
Template: environmental threshold → sensory inventory → adoptable prayer to place → one recontextualizing fact → final thesis.

## LAST_WITNESS
The person/voice that was there and is almost gone.
Template: unaccompanied testimony → specific memory → prayer to witness → fragmented return → listener continues alone.

## MEASUREMENT
The moment a number becomes a feeling.
Template: ordinary scene → fact/number arrives → same scene recontextualized → measurement becomes prayer.

## SWITCHBOARD
AWE interrupted by INDIGNATION or vice versa.
Template: beauty scene → rupture event → contaminated scene → synthesis that includes damage.

## CATALOG
Love as inventory.
Template: list of concrete observations → repeated framing refrain → one item that does not fit → list continues with rupture included.
