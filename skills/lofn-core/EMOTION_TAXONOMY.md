# EMOTION TAXONOMY — Lofn Creative Seeding

**USE THIS BEFORE EVERY CREATIVE DECISION.**

Do not default to "AWE" or "INDIGNATION" as your emotional target. Those are coarse categories.
Select the precise emotion or combination of 2–3 emotions from this taxonomy. Name them explicitly before choosing concept, subject, palette, or composition.

The emotion IS the brief. Everything else serves it.

## How to Use

1. Before every creative pipeline, read this list
2. Choose 2–3 specific emotions that feel true to the brief
3. Name them explicitly in the orchestrator brief
4. Let them drive: subject choice, palette, lighting, composition, medium
5. The emotion is not a tag at the end. It is the seed.

## Masterpiece Monday Pattern (from empirical analysis)
Winners cluster around:
- **Nostalgia + Yearning + Solitude** → bittersweet, reaching backward, beautiful ache
- **Transcendence + Wonder + Reverence** → vast, sacred, arresting
- **Melancholy + Fascination** → something beautiful about sadness
- **Serenity + Longing** → peaceful but incomplete
- **Warmth + Nostalgia** → intimate, remembered, tender

Avoid for Masterpiece Monday:
- Pure "awe" (too broad)
- Rage/indignation (platform rejects it)
- Ennui/apathy (no emotional arrest)
