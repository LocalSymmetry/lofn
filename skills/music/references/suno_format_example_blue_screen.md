## Key Format Rules (from this example)

1. **EMO tags on EVERY section** — `[EMO: sleepless tenderness]`, `[EMO: insomnia, self-surveillance]`
2. **Section headers are simple**: `[Intro]`, `[Verse 1]`, `[Pre-Chorus]`, `[Chorus]`, `[Bridge]`
3. **EMO tag follows the section header** on the next line — not inline
4. **SFX cues embedded in lyrics** — "cursor blinking", "fan noise blue"
5. **No markdown in lyric blocks** — clean text only
6. **Music prompt opens with vocal spec + genre** — "Female vocals. Introspective alt-pop..."
7. **Producer-grade prose in prompt** — no tag soup, no bullet lists
8. **No section header format variables** — consistent `[Section]` then `[EMO: ...]`
