## Key Format Rules (from this example)

1. **First lyric line**: `[Theme: <specific scene-pressure / emotional operating system>]` — NOT a generic topic
2. **Section headers**: `[Section Name – EMO:<emotion(s)> – Vocalist – Instrument cues]`
3. **EMO tags on EVERY section** — never bare `[Verse]` or `[Chorus]`
4. **No markdown formatting** — no bold, no italics inside lyric blocks
5. **No parenthetical prose directions** — cues go in section header brackets
6. **SFX cues** use `*asterisks*` — e.g. `*ice-crunch*`, `*snow settling*`
7. **Vocalist specified** — `Female Vocalist`, with optional modifiers like `+ airy harmonies`, `+ Choir`
8. **Music prompt is prose** — not tag soup, not bullets, not checklists
9. **Music prompt opens with genre/style + tempo/energy** — not "Begin in/by/with..."
10. **No artist names** anywhere in the prompt
