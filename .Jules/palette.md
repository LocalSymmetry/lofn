## 2025-05-15 - Streamlit Button Hierarchy
**Learning:** Streamlit's `type="primary"` combined with Material icons (e.g., `:material/lightbulb:`) significantly improves the discoverability of main actions compared to default buttons.
**Action:** Use `type="primary"` and relevant Material icons for the main "Generate" or "Run" action on each page to establish a clear visual hierarchy.
