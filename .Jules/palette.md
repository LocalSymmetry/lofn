## 2026-01-17 - Contrast Compliance
**Learning:** Default Sky 500 (`#0EA5E9`) used as a primary button background with white text fails WCAG AA contrast (ratio ~3:1). Sky 700 (`#0369A1`) provides a much better ratio (~4.5:1) while staying in the same color family.
**Action:** Use darker shades (Sky 700+) for primary actions with white text.
