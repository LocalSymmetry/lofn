## 2025-05-23 - Streamlit Button Hierarchy
**Learning:** Streamlit apps often suffer from "button soup" where all actions look equal. Using `type='primary'` and Material icons (e.g., `:material/lightbulb:`) significantly improves visual hierarchy and scanability.
**Action:** Default to using icons for all action buttons and reserve `type='primary'` strictly for the main "happy path" action on each page/tab.
