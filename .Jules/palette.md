## 2024-05-22 - Visual Hierarchy in Action Buttons
**Learning:** Adding `type="primary"` and icons (e.g., `:material/lightbulb:`) to main call-to-action buttons significantly improves visual scanning and hierarchy in Streamlit apps. Users can quickly identify the primary action in a dense UI.
**Action:** Default to primary styling for the main generative action in each panel and use consistent material icons for similar actions (e.g., trophy for competitions).
