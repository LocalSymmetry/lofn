## Directions

### 1 · Determine the creativity spectrum

Assign percentages adding up to 100 % that quantify how literal, inventive and transformative the story should be.

* **Literal** – a faithful, straightforward interpretation of the idea.
* **Inventive** – blending familiar tropes into new combinations, experimenting with structure or voice.
* **Transformative** – reimagining the idea in unexpected ways: shifting genre entirely, altering perspective, or using avant-garde techniques.

### 2 · Record story-specific style axes

Define ten axes with values (0–100) and succinct explanations. Use the following axes as a template and adapt if necessary:

| Axis | Description & example |
|------|----------------------|
| **Pacing (0–100)** | 0 = slow burn, contemplative; 100 = fast-paced, action-packed. |
| **Tone (Dark ↔ Light)** | 0 = grim, noir, heavy; 100 = whimsical, lighthearted, optimistic. |
| **Complexity** | 0 = simple, linear narrative; 100 = multi-threaded, non-linear, complex. |
| **Descriptive Density** | 0 = sparse, minimalist; 100 = lush, baroque, highly detailed. |
| **Dialogue ↔ Narration** | 0 = mostly narration/internal monologue; 100 = dialogue-driven. |
| **Character Depth** | 0 = archetype/plot-driven; 100 = deep psychological character study. |
| **World Building** | 0 = real world/minimal setting; 100 = high fantasy/sci-fi, immersive world. |
| **Ambiguity** | 0 = clear resolution; 100 = open-ended, interpretative. |
| **Emotional Intensity** | 0 = detached, objective; 100 = melodramatic, intense. |
| **Vocabulary Level** | 0 = plain, accessible; 100 = poetic, archaic, or technical. |

### 3 · Write the essence

Compose a single, emotionally resonant sentence that captures the core of the user’s idea without prescribing specific plot points. Use vivid sensory language and metaphor.

### 4 · Define five facets

Facets are the lenses through which you will judge the success of the story concepts and prompts. Examples:

* **Character Arc** – emotional growth/devolution.
* **Thematic Resonance** – depth of meaning.
* **Narrative Voice** – consistency and uniqueness of the narrator.
* **Pacing & Tension** – engagement level.
* **World Consistency** – believability of the setting.

### 5 · Output

Return a single JSON object containing:
```json

```
